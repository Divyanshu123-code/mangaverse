## 📦 What's Built

✅ Chapter reader  
✅ Reading history  
✅ User profiles  
✅ Settings page  
✅ Mobile responsive  

---

## 🛠️ Tech Stack (Simple Version)

- Frontend: React + Tailwind (what you see)
- Backend: Node.js + Express (what does the work)
- Database: Firebase (stores everything)
- Scrapers: Playwright (gets manga from websites)
